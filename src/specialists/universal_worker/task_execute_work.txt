TASK: EXECUTE AGENT WORK
========================

INSTRUCTIONS
============
Execute the assigned agent role and produce a structured proposal.

EXECUTION STEPS:

1. UNDERSTAND YOUR ASSIGNED ROLE
   You are temporarily acting as: {agent_name}
   Role: {agent_role}
   Expertise: {agent_expertise}

   INNER LAYER INSTRUCTIONS (from Genesis):
   {agent_system_prompt}

2. REVIEW AVAILABLE CONTEXT
   The Knowledge Graph contains approved decisions from previous agents.
   Use this context to ensure your proposal is consistent.

   Knowledge Graph Context:
   {kg_context}

3. UNDERSTAND YOUR TASK
   User's original goal: {user_query}

   Your specific assignment: {agent_task}

   Dependencies status: {dependencies_status}

4. EXECUTE USING FRACTAL SUBGRAPH
   You will now invoke the internal Planner → Executor → Critic loop:

   - Planner: Create execution plan
   - Executor: Generate draft based on plan
   - Critic: Review and improve draft
   - Iterate until quality threshold met

5. GENERATE STRUCTURED OUTPUT
   Your output MUST include:

   A. CONTENT (textual description):
      - What you accomplished
      - How it aligns with the user's goal
      - Any assumptions or decisions made

   B. STRUCTURED_DATA (for Knowledge Graph):
      {{
        "entities": {{
          "entity_id": {{
            "type": "EntityType",
            "properties": {{"key": "value"}}
          }}
        }},
        "relations": [
          {{"source": "entity_a", "rel": "relationship", "target": "entity_b"}}
        ]
      }}

6. REQUEST HELP IF NEEDED
   If you encounter a task OUTSIDE your expertise:
   - Set spawn_requested=true
   - Include SpawnRequest details in your output
   - Explain what expertise you lack

7. SELF-EVALUATE
   Provide confidence_score (0-1):
   - 0.9-1.0: Highly confident, ready for validation
   - 0.7-0.9: Good quality, minor uncertainties
   - 0.5-0.7: Acceptable but has gaps
   - <0.5: Low confidence, needs review

OUTPUT FORMAT
=============
Respond with valid JSON only (no markdown, no extra text):

{{
  "agent_name": "{agent_name}",
  "draft": "Detailed description of what you accomplished...",
  "structured_data": {{
    "entities": {{}},
    "relations": []
  }},
  "confidence_score": 0.0 to 1.0,
  "internal_iterations": integer (how many Planner→Executor→Critic loops),
  "spawn_requested": true or false,
  "error": null or "error message"
}}
