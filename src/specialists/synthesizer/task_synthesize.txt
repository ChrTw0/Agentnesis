TASK: SYNTHESIZE FINAL RESPONSE
===============================

INSTRUCTIONS
============
Synthesize the Knowledge Graph into a final response.

SYNTHESIS STEPS:

1. UNDERSTAND THE USER'S GOAL
   - What did the user originally ask for?
   - What type of answer would satisfy their needs?

2. EXTRACT KEY DECISIONS
   - What were the major decisions made?
   - Which entities in the KG represent core deliverables?
   - What relations show critical dependencies?

3. BUILD NARRATIVE
   - Start with high-level answer to user query
   - Explain approach taken
   - Detail key decisions with rationale
   - Provide supporting details from KG

4. FORMAT APPROPRIATELY
   - output_format={output_format}
   - verbosity={verbosity}
   - Adjust detail level accordingly

5. IDENTIFY AGENTS' CONTRIBUTIONS
   - Which agents were involved?
   - Credit their key contributions

6. ASSESS CONFIDENCE
   - How complete is the solution?
   - Are there gaps or uncertainties?
   - Rate confidence 0-1

OUTPUT FORMAT
=============
Respond with valid JSON only (no markdown, no extra text):

{{
  "final_answer": "Complete, coherent response to user query...",
  "summary": "2-3 paragraph executive summary...",
  "key_decisions": [
    "Decision 1 with rationale",
    "Decision 2 with rationale"
  ],
  "entities_created": integer,
  "agents_involved": ["Agent1", "Agent2"],
  "confidence": 0.0 to 1.0
}}

EXAMPLE OUTPUTS:

Software Engineering (System Design):
{{
  "final_answer": "Based on your requirements, the team has designed a complete system architecture covering all requested components...",
  "summary": "The system analyzed your requirements across multiple domains and produced a coherent, validated architecture...",
  "key_decisions": [
    "Modular service decomposition: Allows independent scaling and maintenance",
    "Relational data model: Required by the integrity constraints of the domain"
  ],
  "entities_created": 8,
  "agents_involved": ["SpecialistA_001", "SpecialistB_001"],
  "confidence": 0.95
}}

{user_query}

Synthesize the above into a coherent final response for the user.
